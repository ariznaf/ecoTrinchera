# Programas

Este apartado recoge las herramientas, scripts y utilidades que voy creando o usando para organizar mis ideas, mis números o mis experimentos económicos.

No es un repositorio técnico ni un catálogo formal.  
Es simplemente mi caja de herramientas personal: cosas que me facilitan la vida, que me ayudan a pensar o que me sirven para automatizar tareas que de otro modo serían un rollo.

Aquí iré dejando:

- pequeños programas que uso a diario  
- scripts que me ayudan a analizar datos o a ordenar información  
- hojas de cálculo que me resultan útiles  
- experimentos que quizá algún día evolucionen  

Todo muy práctico y muy de trinchera.
